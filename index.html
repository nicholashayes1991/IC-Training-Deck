Hardware Reference

Hardware & Cable Materials

⚡ Power Cabling
C13-C20 cables for server power distribution. Secure with proper strain relief.
🌐 Cat6 Ethernet
Management network connectivity (1Gb/10Gb). Keep runs clean and labeled.
🔗 DAC Cables
High-speed copper (25Gb–400Gb). Minimum 3" bend radius — always.
💡 Fiber Optic
Single/Multi-mode for long-distance connections. Always cap when not in use.
💾 MiniSAS
Storage array connections. Ensure connectors lock fully before continuing.
🗂 Cable Management
Velcro straps only — no zip ties. Gray cables, gentle bends, clean routing.

Step 1 · Build Prep

Build Prep Checklist

Never skip this step — it sets up everything that follows

1
Log into aDB Integration before touching any hardware
2
Verify cabinet configuration — confirm the build matches the work order exactly
3
Inventory all materials — power cables, Cat6/DAC, mounting hardware, cable management supplies
4
Stage tools and equipment — everything in arm's reach before you begin
5
Confirm safety gear is on — PPE, ESD straps, steel-toed shoes
Step 2 · Rack & Stack

Rack and Stack

Installation Steps

  • Install rails per SOP specifications for that cabinet type
  • Mount switches/chassis using proper lifting techniques
  • Secure all equipment with correct hardware
  • Install PDUs in designated locations

Quality Checks

  • Verify alignment and rack unit spacing
  • Document all asset tags in aDB Integration before moving on
  • Confirm all screws are fully tightened — no exceptions
  • Double-check PDU position before starting any cabling
Steps 3 & 4 · Power & Network

Power Installation & Network Cabling

⚡ Power Installation

1
Route power cables through proper management channels
2
Connect to designated PDU outlets per configuration
3
Apply proper strain relief to all connections before moving to Cat6

🌐 Cat6 & DAC Cabling

  • Management network uses Cat6 — keep runs clean and labeled
  • DAC cables: minimum 3" bend radius — always
  • All connectors must be fully flush and seated before dressing
  • Verify correct port assignments before seating any cable
  • Dress with velcro straps only — no zip ties
🧠 Knowledge Check #4

DAC Cable Requirements

When installing DAC cables, what is the minimum bend radius requirement?
A
1 inch
B
2 inches
C
3 inches
D
No minimum requirement
✅ Correct: C — DAC cables require a minimum 3-inch bend radius. Tighter bends can damage copper conductors and cause signal loss or outright cable failure.

Standards Reference

Quality Standards

Every cabinet must meet these — no exceptions, no shortcuts

📐 Physical Requirements

  • Cable bend radius specifications always followed
  • Clean cable dressing — no tangles, no loose runs anywhere
  • All connectors fully seated and locked before dressing
  • Equipment properly aligned and secured in the rack

📋 Documentation Requirements

  • Timely and accurate updates in aDB Integration
  • All asset tags documented before build is marked complete
  • Any deviations logged and reported to FTE Lead immediately
  • Must pass FTE QA inspection and sign-off before next build

Heads Up

Common Challenges

Everyone hits these — here's how to push through them

🔌
Cable Management — Practice proper routing every day. Muscle memory builds fast.
⏰
Time Management — Accuracy before speed. Speed is a byproduct of good habits.
💻
aDB Navigation — Request additional training if you're struggling. That's what it's there for.
✅
Quality Standards — Review specs before each build, not after something looks off.
💬
Communication — Never hesitate to ask. Silence is the most expensive mistake on the floor.
📝
Documentation — Complete it immediately after each task. Memory fades fast on a busy floor.
